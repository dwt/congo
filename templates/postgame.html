What Next?

For the time being, the site will be up and running, so that everyone may view the move sequence on the board, the chat will still be open for use, and the SGF can still be downloaded for viewing on a better (yes, I know) game replay app. Voting and commenting will be disabled.

As promised, the game will be reviewed by Jiang MingJiu 7 Dan Professional. We should have this done within a week; Mr. Jiang's schedule permitting. I will also release some comprehensive statics about this game in a pretty infographic.

The plan going forward is to put the server on hiatus for a couple of weeks while I sort through the backlog of bugs and feature request to fix as much as I can. When you come back, we'll have a shiny revamped version of ConGo!
